PHP's StrTr for C#
This is a String Extension for having the PHP strtr function, which is missing in C# but was very handy in PHP.
It comes as a String.StrTr method with several overloads.

It supports the "two strings" approach for chars to replace as well as the "array" thing.
Instead of the associative array, a List of Tuples or a Dictionary of Key=>Value pairs can be used as first argument.
There is also the option of providing the ("original", "replacement") Tuples directly as method arguments.

As you may see,I enjoy very much having this functionality of strtr back in C#!
