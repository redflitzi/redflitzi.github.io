StrTr for C#
This is a String Extension for having the PHP strtr function, which is mising in C# but was very handy in PHP.

It supports the "two strings" approach for chars to replace as well as the "array" thing.
Instead of the associative array, a List of tuples or a Dictionary of Key=>Value pairs can be used.
